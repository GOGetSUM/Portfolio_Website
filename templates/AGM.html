Automated Garden Mister

It's about to be a HOT summer and to keep things cool, I have created an automated garden mister. Acquiring a garden mister from Amazon was fairly easy and low cost. To save money, this garden mister has been paired with a Kasa smart plug and through the use of API requests and Python Anywhere's scheduling capabilities I have created coded a simple program to run this mister at low maintaince automated status.
In this project the use of Python clients have been used to help build the required code.

  • GeoPy - Used to acquire the required geographical location. (Lat, Lon)
  • Open Weather Map - Used to acquire current weather forecast.
  • Twilio - Used to send SMS notifications
  • Kasa Smart / TP -Link - Kasa Smart Plug hardware was use to control "on/off" modes & status checking of pump.
