Colour Palette

Check this code out! Its an webb application that pulls in a image file and returns a colour palette list in RGB and Hex code of the top five colors used in the image.


The script runs and checks for the actual name of the color pulled from the image, if there is no actual name then closest color name is show.
This might be a tad confusing and a future update might involve superseding actual color with closest color, as this color is the correct color name.


Video:


Repository is set up as usual, html code are found in template, styles/etc. are found in static & main serrver is found on the main folder.


This script leverages the use of Flask,Bootstrap, Colorthief, Webcolors.
These are used to facilitate webpage achistructural build, Image maipulation, Color conversion & libraries.


I devloped the function top_ten() with the intention of returning a pallette of ten objects but adjusted my output to five later in the developement process. This function uses the module Colorthief to extract the top 5 rgb colors and saves them into a list.
I then use a for loop to cycle through each rgb color in the list and convert it into actual rgb color name and hex while saving each value in their respective lists.


HTML code was built for the index web page as well as the top_ten colour palette page.
