Writer's Block App

GUI Application used to help get over writer's block!
This application will delete any work done if the writer stops typing for more than 11 seconds. The screen fades as it approaches the limit.
Don't worry, as a last resort effort I've allowed the information to be saved to file.


Video:

Application Code is built on two sections, user interface and Functions.

Functions:
start_timer()- Begins the timer by actuating check_disappear().
check_disappear()- Main logic function, begins count and checks for limits. Will begin to fade screen as count is greater than 6 seconds.
results() - calculates results, retrieves user paragraph entry at every second, counts words by seperating space as character sets. Outputs live word count.
reset_text()-resets count down mechanism, clears user message box entry and resets results in gui.
