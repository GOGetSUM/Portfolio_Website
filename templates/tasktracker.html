Task Tracker

This web application tracks your to-do tasks.
Keeps a library of the task and status of completion, allows you to search & update each task through the use of SQLalchemy (Database)
The UI is made up of four views, To Do, On-going, Completed List, Search a Task.


Each new task added is given a "Task ID" No. & is shown on the To Do section based on assigned priority. The priority value and ID are immutable and are carried through each view.
The task is then carried through each view based on its completion status, 0-100%


There are three other html pages used in this web application, Add-task, Edit-Task & Search Task.


The search task logic is indexing of a databased on the task ID basis being the key value.

Repository is set up as usual, html code are found in template, styles/etc. are found in static & main serrver is found on the main folder.


This script leverages the use of Flask,WTForms, SQLAlchemy & Bootstrap.

  • Edit
