Watermark App

GUI Application used for watermarking a photo!


The graphical user interface displays a "brand" logo, user instructions, image file path entries, text entry, black or white tick and a submit and quit button.

Video:


The script for this application leverages TK interface, PILLOW (imaging library), Numpy (multi-dimensional arrays and matrices library).
The python script is split into two basic sections GUI-Setup & Functions.


Functions:
watermark()-is used to retrieve user entries, resize image to logo size add water mark if any and save file.
kolor()-is used to adjust text color based on user entry on GUI.
preview(photo)- is used to show the user a preview of the file which was saved. It takes one input as the file path of the photo, pass through watermark() function.
