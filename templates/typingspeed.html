Typing Speed App

GUI Application used for capturing a users words per minute typed in a 30 second span!

The script for this application leverages TK interface & math library.
The GUI of this application explores the config method under TKinter where part of my GUI is responsive to the start of the counter.

Functions:
start_timer()- Begins the timer by actuating countdown() & updates graphical user interface instructions.
count_down(count)- script to count down from a given count entry.
results() - calculates results, retrieves user paragraph entry at end of countdown, counts words by seperating space as character sets.
reset_timer()-resets count down mechanism, clears user message box entry and resets results in gui.
