Online Shop v2

This is a re-visit to the Online shop project I had completed, in this upgrade I use Black to code format the project files, the structure of our databases is also reconfigured, base on best practices. I explore the use of models, schemas and resources for the API endpoints. I learned about database translation and applied new features such as database relationships. We also use a third party emailing service to send confirmation and links and also use Postman to test our API endpoints.
The front end section has been left unchaged from the first version of this project.

This project documentation is composed of 3 - documented videos, narrated by myself.

  • Part 2 - API/Endpoints & Postman Test
