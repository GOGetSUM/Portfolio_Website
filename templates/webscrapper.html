BrainyQuote Scraper to CSV (WebScraper)

This application uses Beautifulsoup to scrape a website for quotes of a specific topic and returns a saved csv of each quote found.


Video:


Noteworthy, this script is set to loop through five different topics and return five seperate csv files
Cfscrape, IO, Random, Pandas & Beautiful Soup has been imported into our project folder.


As explain this script begins with a For loop as to capture the webscraping script hollistacillay and pass through each specific topic.
Requests method has been superseded by CFScrapper which will allow our request to pass through, as BrainyQuote site has included Cloud Flare in their HTML Security.
Once the request has been succesfull, the scirpt then creates a "soup" of the site by using BeautifulSoup's html parser.
This "soup" can then be use to extract the quote and author as text and save into a csv file. To retrieve the quote the html handle selector "div a" is used to finde the required text.
The correspondng author is found by retrieving all the elements in the "soup" which have a title of "view author"
Both the quote list and author list are saved into a dictionary with "Quote" and "Author" as the respective key values.
Pandas module is then used to convert the dictionary into a data frame and then into a csv file.
