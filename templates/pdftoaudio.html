PDF to Audio (HTTPS-AWS)

This application is composed of a GUI which allows the user to enter a pdf file,
the app will then convert this pdf into a mpr and automatically play back what is found.


This script basis works with Amazon Web Services module "Polly" throught the use of their HTTPS AWS services.


Video:


This script imports the following modules TKinter, Pytesseract, PDF2Image,Boto3, Pygame,Time.
The built is simply compose of the GUI section and then a function section for the importing of the image and conversion.


Functions:

  • create_text() - pulls in the pdf file, converts it into text and actuates next function create_speech() while passing through the script.
  • create_speech() - pulls in the script and passes it through "Polly's" synthesizor, plays audio and saves an mpr output file.
