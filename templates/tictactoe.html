Tic Tac Toe Script - 2 Players

Python Script which allows for a 2 player tic tac toe game to take place!


Video:


This script is split into two main functions, a welcome screen & some start code to get things started.
Welcome Screen and User inputs:


Function 1/2:
Move()- Ask for user input, checks if move is valid, then replaces board space with users player character, (X or O)
This script on basis runs through the relation of user's recorded moves (respective to each), board's current que and tic tac toe logic use for the Check() function.
This script also checks for game drawn situations & each player takes a turn using a reference current board view as guidance.
Only 2 player mode is available.


Code repeated for second user based on if/else (odd/even turns) logic.

Function 2/2:
Check()- Is a nested function within Move(), it is responsible for checking current players move against a known dictionary of winning set possibilities.
I built this dictionary based on know winning sets on the tic tac toe board. Trivial.


This logic is interesting and it took some brain juice to discover,
first section checks the number of spots taken on the board by keeping count of each spot that is not found on the current board.
second section check each move based on its know winning set and checks if the set number required are met in the users moves.
